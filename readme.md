## Introduction for installing crawling manga, novel app

1. Download [python](https://www.python.org/downloads/)
2. Create venv enviroment 
```
python -m venv venv
```
3. Start enviroment
```
./venv/Scripts/Activate.ps1 # powershell
./venv/Scripts/active.bat # cmd
```
4. Run install requirements
```
pip install -r requirements.txt
```
